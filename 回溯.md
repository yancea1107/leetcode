## 回溯算法的基本形式是“递归+循环”
### 1. Medium |    17. Letter Combinations of a Phone Number  电话号码的字母组合

https://leetcode-cn.com/problems/letter-combinations-of-a-phone-number/

思路：
* 两个数组
* 遍历上一轮元素 和此轮元素 为新的上一轮元素；

```
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []  
            
        letters = {
            "2": "abc",
            "3": "def",
            "4": "ghi",
            "5": "jkl",
            "6": "mno",
            "7": "pqrs",
            "8": "tuv",
            "9": "wxyz"
        }
        
        res = ['']
        for i in digits:
            tmp = []  # 本轮结果
            for m in res:
                for n in letters[i]:
                    tmp.append(m + n)
            res = tmp
        return res
```       

class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        self.letters = {
            "2": "abc",
            "3": "def",
            "4": "ghi",
            "5": "jkl",
            "6": "mno",
            "7": "pqrs",
            "8": "tuv",
            "9": "wxyz"
        }

        if not digits:
            return []
        self.res = []
        self.helper(digits, 0, [])
        return [''.join(char) for char in self.res]


    def helper(self, digits, index, ans):
        # 如果搜寻到最后一个字符，就找到了一组字母组合，保存
        if index >= len(digits):
            self.res.append(ans)
            return
            
        # 对树形结构进行遍历
        for i in self.letters[digits[index]]:
            self.helper(digits, index + 1, ans + [i])
        return
